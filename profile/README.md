# Spectrum Labs

Spectrum Labs develops and maintains open-source libraries that power our own products and internal platforms.

Rather than building one-off examples, we focus on creating practical, reusable software that solves real-world engineering problems in AI, cloud infrastructure, automation, and developer productivity.

## Projects

### AI Driver

A lightweight provider-agnostic SDK for working with modern AI platforms through a consistent interface.

Supported providers include:

* OpenAI
* Anthropic
* Google Gemini
* Self-hosted inference servers

The library was created to reduce vendor lock-in and simplify AI integrations across our applications and services.

## Our Philosophy

* Build tools we use ourselves
* Keep dependencies minimal
* Prefer simple, composable APIs
* Support multiple providers and platforms
* Open source where it creates value
